# agentic-wormhole

Local-first mobile control plane for coding agents and full-stack apps.

## Install

```bash
npm install -g agentic-wormhole
```

## Usage

```bash
agentic-wormhole serve
agentic-wormhole
agentic-wormhole codex
agentic-wormhole claude
```
